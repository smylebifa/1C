# Примитивные типы данных


## Работа с датами

### Стандартные функции по работе с датами
```
ВыбДата = '20210809';
Месяц(ВыбДата);            // 8
НачалоМесяца(ВыбДата);     // 01.08.2021  0:00:00
КонецДня(ВыбДата);         // 09.08.2021 23:59:59
НеделяГода(ВыбДата);       // 33
ДобавитьМесяц(ВыбДата, 3)  // 09.11.2021  0:00:00
ТекущаяДата()              // 29.10.2021 17:08:50
```

### Получение завтрашнего дня по текущему
```
ТекДень = '20210809112607';
Сообщить(ТекДень);

ТекДата = ТекущаяДата();
ЗавтрашнийДень = ТекДата + 60*60*24;

```

## Работа с строками

### Основные функции по работе с строками
```
ТестоваяСтрока = "   тест   ";
МногострочнаяСтрока = "Первая строка
                      |Вторая строка";

СтрДлина(ТестоваяСтрока);      // 10

СокрЛ(ТестоваяСтрока);         // "тест   "
СокрП(ТестоваяСтрока);         // "   тест"
СокрЛП(ТестоваяСтрока);        // "тест"

Лев(ТестоваяСтрока, 5);        // "   те"
Прав(ТестоваяСтрока, 4);       // "т   "

Сред(ТестоваяСтрока, 4, 4)     // "тест"

СтрНайти(ТестоваяСтрока, "е")   // 5
СтрНайти(ТестоваяСтрока, "ст")  // 6

ВРег(ТестоваяСтрока) // "   ТЕСТ   "
ТРег(ТестоваяСтрока) // "   Тест   "

СтрЗаменить(ТестоваяСтрока, "е", "о") // "   тост   "

СтрЗаменить("   те ст    ", " ", "")  // "тест"

СтрЧислоСтрок(МногострочнаяСтрока) // 2

СтрПолучитьСтроку(МногострочнаяСтрока, 2) // "Вторая строка"
```

### Пример - определяем является ли строка числом
```
ВходнаяСтрока1 = "8609.47";
ВходнаяСтрока2 = "8609.47%";

СтрокаЦифр = "0123456789.";
КоличествоСимволов = СтрДлина(ВходнаяСтрока1);
ЭтоЧисло = Истина;

Для Счетчик = 1 По КоличествоСимволов Цикл
  ТекСимвол = Сред(ВходнаяСтрока1, Счетчик, 1);
  РезультатПоиска = СтрНайти(СтрокаЦифр, ТекСимвол);
  Если РезультатПоиска = 0 Тогда
    ЭтоЧисло = Ложь;
    Прервать; 
  КонецЕсли;
КонецЦикла; 

Если ЭтоЧисло = Истина Тогда
  Сообщить("Строка является чилом");
Иначе 
  Сообщить("Строка не является чилом");
КонецЕсли;


СтрРазделить("синий, желтый", ",") // [0-1] = "синий", "желтый"
```

### Пример - получение дня недели (в виде строки)

```
ТекДата = НачалоМесяца(ТекущаяДата());
КонецМесяца = КонецМесяца(ТекущаяДата());

Пока ТекДата <= КонецМесяца Цикл

    Если НомерДня = 1 Тогда
      ДеньНеделиСтрокой = "Понедельник";    
    
    ИначеЕсли НомерДня = 2 Тогда
      ДеньНеделиСтрокой = "Вторник";    
    
    ИначеЕсли НомерДня = 3 Тогда
      ДеньНеделиСтрокой = "Среда";
    
    ИначеЕсли НомерДня = 4 Тогда
      ДеньНеделиСтрокой = "Четверг";    

    ИначеЕсли НомерДня = 5 Тогда
      ДеньНеделиСтрокой = "Пятница";

  КонецЕсли;

  ТекДата = ТекДата + 60*60*24;

КонецЦикла; 
```


## Правила преобразования типов

> Приведение типа происходит относительно первого операнда
```
2 + "2" = 4

"2" + 2 = "22"
```
